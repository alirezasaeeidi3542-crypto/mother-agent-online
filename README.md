# Mother Agent Dr.saeeidi

پروژه فرانت‌اند + اسکریپت دیتابیس برای Mother Agent Dr.saeeidi

## ساختار
- frontend/: کد React و UI
- database/: اسکیمای SQL

## اجرا لوکال (فرانت)
1. cd frontend
2. npm install
3. npm run dev

## نکات امنیتی
کلیدهای حساس را در GitHub Secrets یا فایل `.env` **ثبت نکنید** در ریپو عمومی.
